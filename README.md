## 🧰 Scripts Included

### 🔐 System Information & Recon Menu (`menu.sh`)

* Check present working directory
* Monitor system uptime
* Audit home directory
* Display IP configuration
* View disk partitions using `lsblk`

---

## ▶️ How to Run

```bash
# Give permission
chmod +x menu.sh

# Run script
./menu.sh
```
